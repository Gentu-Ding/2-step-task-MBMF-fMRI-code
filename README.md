# 2-step-task-MBMF-fMRI-code

This repository contains the task code (MATLAB PTB) to run the 2-step task and the analysis code on the behavioral and fMRI data to conduct the model-based fMRI analysis.

Note: Within this repository, the author of the task code and the computational model is Dr. Jeffrey Cockburn. 

Please also refer to the following preprints (publication in press) for relevant task and analysis information:
1. Ding et al. (2025), https://doi.org/10.1101/2025.09.17.676398; 
2. Cockburn et al. (2024), https://doi.org/10.31234/osf.io/egq3x.
